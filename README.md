# 📊 Python Data Analysis Project

## 📌 Project Title
E-Commerce Sales Performance Analysis

## 📖 Description
This project analyzes e-commerce sales data to understand business performance and customer behavior.  
It helps in identifying:
- Top selling products
- Monthly sales trends
- Revenue patterns

## 🛠 Tools & Technologies Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## 📂 Files Included
- ECOMMERS SALES (1).csv
- Untitled13.ipynb

## 📊 Key Analysis Performed
- Data cleaning and preprocessing
- Handling missing values and duplicates
- Exploratory Data Analysis (EDA)
- Sales trend visualization using charts

## 📈 Insights
- Identified best performing products
- Observed monthly sales variations
- Understood revenue distribution patterns

## 🚀 How to Run the Project
1. Open Jupyter Notebook
2. Load `Untitled13.ipynb`
3. Ensure CSV file is in the same folder
4. Run all cells step by step
